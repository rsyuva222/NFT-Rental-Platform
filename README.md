# Project Title
NFT Rental Platform (Soroban Smart Contract)

# Project Description
This project implements a simple NFT Rental Platform using the Soroban SDK on the Stellar blockchain. It allows an NFT owner to list their NFT for rent, a user to rent it for a fixed duration, and the owner to reclaim it after the rental period ends.

# Project Vision
The vision of this project is to introduce the concept of temporary ownership of digital assets (NFTs), enabling users to utilize NFTs without permanently owning them. This can be useful in gaming, digital art access, memberships, and virtual assets.

# Key Features
- List NFT for rent with price and duration
- Rent NFT for a specified time period
- Automatic validation of rental duration
- Owner reclaim functionality after rental expiry
- View current rental status

# Future Scope
- Support for multiple NFTs instead of a single rental record
- Integration with real NFT token standards
- Payment handling using Stellar tokens
- Role-based access control (Owner/Admin)
- Expiry automation using cron-like triggers
- Frontend dApp integration for real-world usage
